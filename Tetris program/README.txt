Source code files:
button.py - 
ButtonCommand.py - 
check.py - functions for checking i.e. check for loss and check if blocks can move into a certain tile
factory.py - factory file for creating blocks
GameController.py - the game controller
other.py - other miscellaneous functions
score.py - file for containing score related functions
settings.py - file for the configuration page
StartUp.py - the start up page
TetrisGame.py - the game page

Naming convention:
classes in first letter capital cases no spaces e.g. GameSettings
methods in lowercase with underscores e.g. get_rotation()
variable names in lower case with underscores e.g. x_coordinates
